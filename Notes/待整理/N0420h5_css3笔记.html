什么是 html5

  • HTML5 是最新的 HTML 标准。
  • HTML5 是专门为承载丰富的 web 内容而设计的，并且无需额外插件。
  • HTML5 拥有新的语义、图形以及多媒体元素。
  • HTML5 提供的新元素和新的 API 简化了 web 应用程序的搭建。
  • HTML5 是跨平台的，被设计为在不同类型的硬件（PC、平板、手机、电视机等等）之上运行。

HTML5 新特性

  • 新的语义元素，比如 <header>, <footer>, <article>, and <section>。
  • 新的表单控件，比如数字、日期、时间、日历和滑块。
  • 强大的图像支持（借由 <canvas> 和 <svg>）
  • 强大的多媒体支持（借由 <video> 和 <audio>）
  • 强大的新 API，比如用本地存储取代 cookie。

HTML5 新增语义标签

  • <header> —— 定义文档或节的页眉。
  • <nav> —— 定义文档内的导航链接。
  • <article> —— 定义文档内的文章。
  • <section> —— 定义文档中的节。
  • <aside> —— 定义页面内容之外的内容。
  • <footer> —— 定义文档或节的页脚。
注意：
1、语义化标准主要针对搜索引擎
2、ie9中，需转化为块级元素（所有浏览器，包括旧的和最新的，对无法识别的元素会作为内联元素自动处理。）
3、移动端更喜欢用这些标签（移动端没兼容性问题）

属性选择器

为带有特定属性的 HTML 元素设置样式
选择符 简介 实例
element[attribute] 选取带有指定属性的元素。 
                    a[target] {
                        background-color: yellow;
                      }
element[attribute="value"] 【重要！】 选取带有指定属性和值的元素。 
                    a[target="_blank"] { 
                        background-color: yellow;
                      }
[attribute~="value"] 选取属性值 包含 指定词的元素。 
                    [title~="flower"] {
                        border: 5px solid yellow;
                      }
[attribute|="value"] 选取 指定属性 以 指定值开头 的元素 
                    [class|="top"] {
                        background: yellow;
                      }
                    值 必须是完整或单独的单词, 如： topcontent不算
[attribute^="value"] 选取 指定属性 以 指定值开头 的元素。 
                    [class^="top"] {
                        background: yellow;
                      }
                      值 不必是完整单词！, 如： topcontent也算
[attribute$="value"] 选取 指定属性 以 指定值结尾 的元素。 
                    [class$="test"] {
                        background: yellow;
                      }
                      值 不必是完整单词！, 如： mytest也算
[attribute*="value"] 选取 属性值 包含指定词 的元素。 
                    [class*="te"] {
                        background: yellow;
                      }
                      值 不必是完整单词

根据文档结构来选择元素，常用于根据父级选择里面的子元素
伪类选择器分类
  • 状态类： :link :visited :hover :active :focus
  • 结构类： :first-child :last-child :nth-child :nth-of-type
  • 表单类： :checked :disabled :vaild :required
  • 语言类： :dir :lang
CSS类选择器:伪类选择器
例如：
a:hover
p:first-child

选择符 简介
son element:first-child 匹配父元素中的第一个子元素
son element:last-child 匹配父元素中的最后一个子元素
son element:nth-child(n) 匹配父元素中的第 n 个子元素
element:first-of-type 指定类型e的第一个
element:last-of-type 指定类型e的最后一个
element:nth-of-type(n) 指定类型e的第 n 个

nth-child（n）
  • n 可以是数字，关键字和公式
  • n 如果是数字，就是选择第n个子元素，里面数字从1开始…
  • n 可以是 [关键字] ：[ 奇 odd / 偶 even ] ；
  • n 可以是 [公式] ： 常见公式如下：
    如果 n 是公式，则从0开始计算，第0个元素和超出元素会被忽略。 全选
    公式 取值
    2n 偶数
    2n+1 奇数
    5n 5 10 15 ……
    n 5~最后一个（包含第5个）
    -n+5 前五个（包含第5个）
nth-child(n) 选择特性：
1、执行时会把所有子盒子排列序号
2、先选定 n ，在看类选择器是否匹配，不匹配则不选

nth-fo-type（n）选择特性：
1、执行时只会把指定type的盒子排列序号
2、再选定n

小结
  • 结构伪类选择器一般用于选择父级里面的第几个子级
  • nth-child 对父元素中所有子元素排序，先找第 n 个子元素，再看与类选择器是否匹配
  • nth-of-type 对父元素中指定子元素排序，先匹配类选择器，再找第n个子元素
  • 无序列表中， nth-child 更多（都是li）
  • 类选择器、属性选择器、伪类选择器，权重都为10

CSS3 transition 过渡

CSS 过渡允许您在给定的时间内平滑地改变属性值
如需创建过渡效果，必须明确两件事：
  • 您要添加效果的 CSS 属性(默认为0,，不设置则没效果)
  • 效果的持续时间(默认为0,，不设置则没效果)
指定的 CSS 属性（width）值发生变化时，将开始过渡效果。
属性 简介 实例
transition 也是简写属性
transition:要过渡属性（所有可写all） 过渡时间（必须） 运动曲线 延迟触发时间		 
                    .father {
                        width: 300px;
                        height: 200px;
                        background-color: pink;
                        transition: width 1s, height 1s;
                    }
                    .father:hover {
                        width: 350px;
                        height: 250px;
                    }
transition-timing-function 规定过渡效果的速度曲线。
属性：
  • ease - 规定过渡效果，先缓慢地开始，然后加速，然后缓慢地结束（默认）
  • linear - 规定从开始到结束具有相同速度的过渡效果
  • ease-in -规定缓慢开始的过渡效果
  • ease-out - 规定缓慢结束的过渡效果
  • ease-in-out - 规定开始和结束较慢的过渡效果
  • cubic-bezier(n,n,n,n) - 允许您在三次贝塞尔函数中定义自己的值
transition-delay 规定过渡效果启动前的延迟（以秒计） 
                    div {
                        transition-delay: 1s;
                      }
transition-duration 规定过渡效果要持续多少秒或毫秒。
transition-property 规定过渡效果所针对的 CSS 属性的名称。

CSS 2D 转换

CSS 转换（transforms）允许您移动、旋转、缩放和倾斜元素。
属性 简介 实例
translate() 从其当前位置移动元素（根据为 X 轴和 Y 轴指定的参数）。
百分比是基于盒子自身		 
                    div {
                        transform: translate(50px, 100px);
                      }
rotate() 根据给定的角度顺时针或逆时针旋转元素 
                    div {
                        transform: rotate(-20deg);
                      }
scale() 增加或减少元素的大小（根据给定的宽度和高度参数）。
不会影响其他盒子，且可以设置缩放中心点		 
                    div {
                        transform: scale(0.5, 0.5);
                      }
scaleX() 增加或减少元素的宽度。 
                    div {
                        transform: scaleX(2);
                      }
scaleY() 增加或减少元素的高度。 
                    div {
                        transform: scaleY(0.5);
                      }
skew() 使元素沿 X 轴和 Y 轴倾斜给定角度。
如果未指定第二个参数，则值为零。		 
                    div {
                        transform: skewX(20deg);
                      }
skewX() 使元素沿 X 轴倾斜给定角度。 
                    div {
                        transform: skewX(20deg);
                      }
matrix() 把所有 2D 变换方法组合为一个。
参数如下：matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY())
transform-origin 允许你改变被转换元素的位置。 默认值 50% 50%

2D 转换 综合写法

  • 同时使用多个转换，格式:transform:translate() rotate() scale()等
  • 其顺序会影响转换的效果。（先旋转会改变坐标轴方向）
  • 位移放在最前

CSS 动画

CSS 可实现 HTML 元素的动画效果，而不使用 JavaScript 或 Flash！
什么是 CSS 动画？
  • 动画使元素逐渐从一种样式变为另一种样式。
  • 您可以随意更改任意数量的 CSS 属性。
  • 如需使用 CSS 动画，您必须首先为动画指定一些关键帧。
  • 关键帧包含元素在特定时间所拥有的样式。
如何使用
  • 先定义动画 @keyframes name
  • 再调用动画 animation-name + animation-duration
@keyframes 规则
  • 在 @keyframes 规则中指定了 CSS 样式，动画将在特定时间逐渐从当前样式更改为新样式。
  • 要使动画生效，必须将动画绑定到某个元素。
  • animation-duration 属性定义需要多长时间才能完成动画,默认为0s，必须指定，否则动画不发生
  • 关键字 "from" 和 "to"（代表 0％（开始）和 100％（完成））
  • 可以使用百分比值:0% 25% 50% 100%
代码示例： 
            div {
                width: 250px;
                height: 34px;
                border: 2px solid palegreen;
                background-color: aqua;
                transition: all .7s;
                animation-name: text;
                animation-duration: 3s;
            }
            @keyframes text {
                from {
                    background-color: aqua;
                    float: left;
                }
                25% {
                    background-color: blueviolet;
                    float: right;
                }
                50% {
                    background-color: rgb(21, 171, 118);
                    float: right;
                }
                to {
                    background-color: rgb(222, 101, 30);
                    float: left;
                }
            }
            或
            div {
                width: 250px;
                height: 34px;
                border: 2px solid palegreen;
                background-color: aqua;
                transition: all .7s;
                animation-name: text;
                animation-duration: 3s;
            }
            @keyframes text {
                0% {
                    background-color: aqua;
                    float: left;
                }
                25% {
                    background-color: blueviolet;
                    float: right;
                }
                50% {
                    background-color: rgb(21, 171, 118);
                    float: right;
                }
                100% {
                    background-color: rgb(222, 101, 30);
                    float: left;
                }
            }
属性 说明 实例
animation-name 规定 @keyframes 动画的名称。
@keyframes 规定动画模式。
animation 设置所有动画属性的 简写属性 。格式：animation: name duration timing-function delay iteration-count direction [名时速延次方] 
                    div {
                        animation: example 5s linear 2s infinite alternate;
                      }
animation-delay 规定动画开始的延迟时间。如果使用负值，则动画将开始播放，如同已播放 N 秒。 
                    div {
                        width: 250px;
                        height: 34px;
                        border: 2px solid palegreen;
                        background-color: aqua;
                        transition: all .7s;
                        animation-name: text;
                        animation-duration: 3s;
                        animation-delay: 1s;
                    }
animation-iteration-count 指定动画应运行的次数。使用值 "infinite" 动画将永远持续下去 
                    div {
                        width: 250px;
                        height: 34px;
                        border: 2px solid palegreen;
                        background-color: aqua;
                        transition: all .7s;
                        animation-name: text;
                        animation-duration: 3s;
                        animation-iteration-count: 3;
                    }
animation-direction 定是向前播放、向后播放还是交替播放动画。
  • normal - 动画正常播放（向前）。默认值
  • reverse - 动画以反方向播放（向后）
  • alternate - 动画先向前播放，然后向后
  • alternate-reverse - 动画先向后播放，然后向前
animation-timing-function 规定动画的速度曲线。
  • ease - 指定从慢速开始，然后加快，然后缓慢结束的动画（默认）
  • linear - 规定从开始到结束的速度相同的动画
  • ease-in - 规定慢速开始的动画
  • ease-out - 规定慢速结束的动画
  • ease-in-out - 指定开始和结束较慢的动画
  • cubic-bezier(n,n,n,n) - 运行您在三次贝塞尔函数中定义自己的值
animation-fill-mode 规定在不播放动画时目标元素的样式。 
                    
    
                        
  • none - 默认值。动画在执行之前或之后不会对元素应用任何样式。
    • 
                        
  • forwards - 元素将保留由最后一个关键帧设置的样式值（依赖 animation-direction 和 animation-iteration-count）。
    • 
                        
  • backwards - 元素将获取由第一个关键帧设置的样式值（取决于 animation-direction），并在动画延迟期间保留该值。
    • 
                        
  • both - 动画会同时遵循向前和向后的规则，从而在两个方向上扩展动画属性。

CSS Flexbox

在 Flexbox 布局模块（问世）之前，可用的布局模式有以下四种：
  • 块（Block），用于网页中的部分（节）
  • 行内（Inline），用于文本
  • 表，用于二维表数据
  • 定位，用于元素的明确位置
弹性框布局模块，可以更轻松地设计灵活的响应式布局结构，而无需使用浮动或定位。
flexbox 使用
  • 定义 flex-container

flex-container 属性

属性 属性值 说明 实例
flex-direction
  • row ~ 水平堆叠
  • row-reverse ~ 水平堆叠（但从右到左）
  • column ~ 垂直堆叠
  • column-reverse ~ 垂直堆叠 flex 项目（但从下到上）
 定义容器要在哪个方向上堆叠 flex 项目。

  • nowrap ~ 规定将不对 flex 项目换行（默认）
  • wrap ~ 规定 flex 项目将在必要时进行换行
  • wrap-reverse ~ 规定如有必要，弹性项目将以相反的顺序换行
 是否应该对 flex 项目换行

  • space-between ~ 显示行之间有空格的 flex 项目(项目间 n-1 个等距间隙)

  • flex-end ~ flex 项目在容器底部对齐

  • stretch ~ 拉伸弹性线以占据剩余空间（默认）

                align-items: center;

媒体查询 Media Query

媒体查询可用于检查许多事情，例如：
  • 视口的宽度和高度
  • 设备的宽度和高度
  • 方向（平板电脑/手机处于横向还是纵向模式）
  • 分辨率

媒体查询语法

            @media not|only mediatype and (expressions) {
                CSS-Code;
              }
除非您使用 not 或 only 运算符，否则媒体类型是可选的，且隐含 all 类型。
实例： 
            @media screen and (min-width: 480px) {
                body {
                  background-color: lightgreen;
                }
              }
精确使用 992px 和 600px 吗？它们就是我们所称的设备的“典型断点”（typical breakpoints）。

常用用法

  • 菜单的媒体查询 ~ 创建响应式导航菜单，在不同的屏幕尺寸上会有所不同
  • 列的媒体查询 ~ 常见用法是创建弹性布局，布局具体取决于不同的屏幕尺寸
  • 用媒体查询隐藏元素 ~ 在不同屏幕尺寸上隐藏元素
  • 用媒体查询改变字体 ~ 更改不同屏幕尺寸上的元素的字体大小
  • 弹性图片库 ~ 将媒体查询与 flexbox 一起使用来创建响应式图片库
  • 弹性网站 ~ 将媒体查询与 flexbox 一起使用，以创建响应式网站
  • 方向：人像 / 风景 ~ 根据浏览器的方向更改页面的布局
  • 最小宽度到最大宽度 ~ 使用 max-width 和 min-width 属性设置最小宽度和最大宽度
